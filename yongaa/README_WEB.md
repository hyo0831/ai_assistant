# 🌙 ORION 투자 성향 진단 서비스 - 웹 버전

**밤하늘의 사냥꾼, 당신의 투자 본능을 분석합니다**

ORION은 AI 기반의 투자 성향 진단 시스템입니다. 간단한 설문을 통해 당신의 투자 스타일을 분석하고, 별자리를 테마로 한 맞춤형 성향 닉네임과 조언을 제공합니다.

## 🚀 시작하기

### 1단계: 필수 패키지 설치

```bash
cd yongaa
pip install -r requirements.txt
```

### 2단계: Flask 웹 서버 실행

```bash
python app.py
```

웹 브라우저에서 `http://localhost:5000` 으로 접속하세요.

## 📋 기능

### ✨ 투자 성향 진단
- **시장 선택**: 국내주식, 해외주식, 암호화폐 중 선택
- **10문항 설문**: 당신의 실제 투자 성향 분석
- **AI 분석**: GPT-3.5-turbo로 맞춤형 분석 (저비용)
- **성향 닉네임**: 별자리 + 동물 조합으로 创意적인 유형 분류

### 🎯 9가지 성향 유형

#### 📈 국내 주식 (3가지)
- **리겔의 매** - 돌파 단타형: 거래량 폭발에 빠르게 반응
- **큰곰자리의 곰** - 스윙 안정형: 눌림목에서 묵직하게 진입
- **카시오페이아 여우** - 역발상형: 테마의 시작을 남보다 먼저 포착

#### 🌍 해외 주식 (3가지)
- **아퀼라의 독수리** - 모멘텀 기술주형: 나스닥 상공의 모멘텀 사냥꾼
- **처녀자리 올빼미** - 펀더멘털 분석형: 밤새 재무제표를 분석하는 관찰자
- **남십자성 고래** - 매크로 ETF형: 거시적 흐름을 읽는 투자자

#### ₿ 암호화폐 (3가지)
- **전갈자리 치타** - 레버리지 단타형: 선물로 빠르고 치명적인 수익
- **황소자리 황소** - 현물 홀딩형: BTC·ETH를 믿고 버티는 장기 보유자
- **쌍둥이자리 카멜레온** - 스윙 적응형: 상승장·하락장 모두 대응하는 멀티플레이어

## 📝 사용 방법

### 1️⃣ 시장 선택
국내주식, 해외주식, 암호화폐 중 하나를 선택합니다.

### 2️⃣ 설문 작성 (상세)

설문은 **객관식 + 주관식**으로 구성됩니다:

**객관식 문항:**
- 주어진 선택지 중에서 선택합니다
- 📌 **중복 선택 가능**: 여러 개를 동시에 선택할 수 있습니다
- 본인의 생각과 가장 가까운 것들을 모두 체크하세요

**주관식 문항:**
- 자유롭게 직접 의견을 작성합니다
- 구체적이고 솔직한 답변이 정확한 분석의 핵심입니다
- 예시: "손실을 보면 즉시 판다" / "기술분석 중심으로 투자한다" 등

### 3️⃣ 분석 결과 확인
- 당신의 성향 닉네임 (별자리 + 동물)
- 성향 상세 분석
- 강점과 약점
- 맞춤형 조언

## 📞 문제 해결

### ".env 파일에 OPENAI_API_KEY를 설정하세요" 오류
- `yongaa/.env` 파일이 존재하는지 확인하세요
- 파일 내용:
  ```
  OPENAI_API_KEY=sk-proj-your_actual_key_here
  ```
- API 키 값이 올바르게 입력되었는지 확인하세요

### "API 키 오류" 발생
- `.env` 파일의 API 키를 다시 확인하세요
- OpenAI 계정의 크레딧이 충분한지 확인하세요
- https://platform.openai.com/api-keys 에서 새 키를 생성해보세요

### "설문 로드 실패"
- 브라우저 콘솔(F12)을 열어 에러를 확인하세요
- Flask 서버가 정상 실행 중인지 확인하세요

### "분석 중 오류"
- 모든 설문 항목을 선택했는지 확인하세요
- `.env` 파일의 API 키가 정상인지 확인하세요
- Flask 서버 로그에서 에러 메시지를 확인하세요

## 📚 추가 정보

### ORION 콘셉트
ORION은 밤하늘의 사냥꾼입니다. 이 콘셉트에서 영감을 받아:
- 별자리(Constellation): 다양한 투자자의 특성을 별자리로 표현
- 동물(Animal): 각 별자리의 상징 동물로 성향을 직관적으로 표현
- 사냥법(Hunting Style): 각자의 투자 스타일을 "사냥법"으로 표현

### GPT-3.5-turbo의 장점
- **저비용**: 입력 $0.50/1M토큰, 출력 $1.50/1M토큰
- **빠른 응답**: GPT-4보다 훨씬 빠른 응답 속도
- **충분한 성능**: 설문 분석에는 충분한 능력
- **안정적**: 오랫동안 검증된 모델
