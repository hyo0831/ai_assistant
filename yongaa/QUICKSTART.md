## 5분 안에 시작하기

### 단계 1️⃣: 패키지 설치 (1분)

```bash
cd yongaa
pip install flask openai
```

### 단계 2️⃣: 웹 서버 실행 (10초)

```bash
python app.py
```

출력:
```
 * Running on http://127.0.0.1:5000
```

### 단계 3️⃣: 웹 브라우저에서 접속

브라우저 주소창에 다음을 입력:
```
http://localhost:5000
```

### 단계 4️⃣: API 키 설정 완료

API 키는 서버 관리자가 이미 `.env` 파일에 설정했습니다. 사용자는 별도로 할 일이 없습니다.

## 🆘 자주 묻는 질문

### Q: "ModuleNotFoundError: No module named 'flask'" 오류
**A**: `pip install flask python-dotenv` 를 실행하세요.

### Q: 포트 5000이 이미 사용 중이라는 오류
**A**: `app.py`의 마지막 줄을 수정하세요:
```python
app.run(debug=True, port=5001, host='0.0.0.0')  # 5000을 5001로 변경
```

### Q: API 키는 안전한가?
**A**: 네! ORION은 다음과 같이 보안을 지킵니다:
- API 키는 `.env` 파일에만 저장
- 사용자 브라우저로 API 키를 전송하지 않음
- `.gitignore`로 Git에 커밋되지 않음
- API 키 값이 로그에 노출되지 않음

### Q: 모바일에서도 사용 가능한가?
**A**: 네! 반응형 디자인으로 모바일에서도 완벽히 작동합니다.
